# 预置指标体系快速入门

## 5分钟快速上手

### 第一步：初始化指标体系（30秒）

1. 打开任意项目的项目详情页
2. 点击"任务管理"标签
3. 点击"加载预置指标体系"按钮
4. 在弹窗中点击"确认初始化"

✅ **完成！** 系统已为您创建17个指标和6个竞品数据

### 第二步：创建任务并关联指标（2分钟）

1. 点击"创建任务"按钮
2. 填写任务基本信息：
   - 任务名称：例如"电池包能量密度优化"
   - 任务类型：选择"仿真"
   - 负责人：选择团队成员
   - 时间：设置开始和截止日期

3. **关键步骤** - 选择关联指标：
   - 切换到"指标列表"标签
   - 展开"性能指标"类别
   - 勾选"续航里程"和"百公里能耗"
   - 或切换到"雷达图对比"标签查看竞品对比

4. 点击"创建任务"

✅ **完成！** 任务已创建并关联指标

### 第三步：查看雷达图对比（1分钟）

1. 在任务创建对话框中，点击"雷达图对比"标签
2. 选择"性能指标"维度
3. 查看蓝色区域（项目目标）与其他颜色区域（竞品）的对比
4. 识别优势指标（蓝色区域突出的部分）
5. 识别差距指标（其他颜色突出的部分）

✅ **完成！** 您已了解项目与竞品的指标差距

### 第四步：评估任务指标（1分钟）

1. 在任务列表中找到刚创建的任务
2. 点击任务卡片打开详情
3. 在"指标评估"面板中填写：
   - 来源：选择"客户需求"
   - 完成信心：选择"高"
   - 信心依据："基于现有电池技术路线"
   - 最优性：选择"接近最优"
   - 目标值：600km

4. 点击"保存评估"

✅ **完成！** 指标评估已记录，建立了MBSE追溯链

## 核心概念

### 什么是预置指标体系？

预置指标体系是系统内置的17项汽车行业核心指标和6个主流竞品的对标数据，涵盖性能、成本、可靠性、安全、合规5大类别。

### 为什么需要指标体系？

在MBSE（基于模型的系统工程）中，指标是连接需求、模型和验证的桥梁：

```
客户需求 → 项目指标 → 任务指标 → 模型参数 → 工作流验证 → 结果确认
```

### 雷达图有什么用？

雷达图将不同量纲的指标归一化到0-100分制，在一张图上对比：
- 项目目标 vs 竞品实际表现
- 识别优势指标（我们领先的地方）
- 识别差距指标（需要重点攻关的地方）

## 典型使用场景

### 场景1：新项目立项

**目标**: 快速建立项目指标基准

```
1. 初始化预置指标体系
2. 查看雷达图，识别与竞品的差距
3. 调整指标目标值（如果需要）
4. 创建Phase 1的关键任务，关联核心指标
```

### 场景2：任务派发

**目标**: 将项目指标分解到具体任务

```
1. 创建任务（如"电池包设计优化"）
2. 关联相关指标（续航、能耗、成本）
3. 评估指标可行性和信心度
4. 派发给工程师
```

### 场景3：竞品对标分析

**目标**: 了解市场竞争态势

```
1. 打开雷达图对比
2. 选择"性能指标"维度
3. 对比项目目标与特斯拉、比亚迪等竞品
4. 识别需要提升的短板指标
5. 创建针对性的改进任务
```

### 场景4：设计方案权衡

**目标**: 在冲突指标间找平衡

```
1. 查看"性能vs成本"雷达图
2. 发现续航提升会增加电池成本
3. 在"权衡研究"模块创建分析
4. 设定优化目标和约束
5. 找到帕累托最优方案
```

## 常见问题

**Q: 指标目标值可以修改吗？**
A: 可以。在"指标管理"中编辑指标，修改目标值、上下限等。

**Q: 可以添加自定义指标吗？**
A: 可以。点击"创建指标"添加项目特定的指标。

**Q: 雷达图中的分数是怎么算的？**
A: 所有指标归一化到0-100分，综合得分是各指标得分的平均值。

**Q: 竞品数据准确吗？**
A: 预置数据来自官方和权威媒体，但建议根据最新信息更新。

**Q: 任务必须关联指标吗？**
A: 不强制，但建议关联以建立完整的MBSE追溯链。
